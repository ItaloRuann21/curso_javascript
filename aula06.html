Revisão

Aprendemos:

  • Comentários por linha utilizando "//"

  • Comentários por bloco utilizando "/* */"

  • Variáveis usando let ou var
    • let produto = 'Arroz'
    • var preco = 7.77

  • Duas maneiras de transformar o preco em vírgula (valor brasileiro)
    • console.log(new Int1.NumberFormat('pt-br').format(variavelAqui))
    • document.write(`Preço total ${variavel.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'})}`)

  • Criar constantes (Valores que não podem ser alterados)
    • const nome = 'Edson Maia'
    • const peso = 83.5
    • const altura = 1.75
    • const imc = peso / (altura * altura)

  • Utilizar método para diminuir quantidade de casas decimais com toFixed()
    • let compra = 2000.99
    • let desconto = 200.99
    • let compraComDesconto = compra - desconto
    • console.log(compraComDesconto.toFixed(2))
    • saída: 1800.00
